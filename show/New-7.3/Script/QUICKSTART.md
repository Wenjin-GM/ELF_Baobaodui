# 智能安全工具柜触摸屏界面 - 快速开始指南

## 🎯 项目简介

这是宝宝队参加 2026 年 Rockchip / Forlinx ElfBoard 嵌入式竞赛的作品——**基于 RK3588 的电网运维智能安全工具柜系统**的触摸屏界面程序。

## 📦 文件结构

```
Show/Script/
├── main.py                 # ⭐ 程序入口，从这里启动
├── main_window.py          # 主窗口（页面切换、导航栏）
├── state_machine.py        # 状态机（9种系统状态）
├── mock_backend.py         # 模拟数据后端
├── config.py               # 配置文件
├── pages/                  # 页面模块目录
│   ├── __init__.py
│   ├── dashboard_page.py   # 总览页
│   ├── auth_page.py        # 认证页
│   ├── tools_page.py       # 工具盘点页
│   ├── charging_page.py    # 充电管理页
│   ├── environment_page.py # 环境监控页
│   ├── records_page.py     # 记录查询页
│   ├── settings_page.py    # 设置页
│   └── debug_page.py       # 调试维护页
├── README.md               # 详细文档
├── SUMMARY.md              # 开发总结
├── run.sh                  # Linux 启动脚本
└── run.bat                 # Windows 启动脚本
```

## 🚀 快速启动

### Windows 用户

1. 双击 `run.bat` 即可启动
2. 或在命令行执行：
   ```cmd
   cd Show\Script
   python main.py
   ```

### Linux / ELF2 开发板用户

1. 运行启动脚本：
   ```bash
   cd Show/Script
   chmod +x run.sh
   ./run.sh
   ```

2. 或直接运行：
   ```bash
   cd Show/Script
   python3 main.py
   ```

## 🎮 操作指南

### 首次启动

1. 程序启动后显示**总览页**（未认证待机状态）
2. 观察数据实时更新：
   - 环境数据每 1 秒更新
   - 工具状态每 2 秒更新
   - 充电状态每 3 秒更新

### 模拟认证流程

1. 点击底部导航栏的 **"认证"** 按钮
2. 点击 **"模拟管理员认证"** 或 **"模拟普通用户认证"**
3. 认证成功后自动返回总览页
4. 导航栏自动更新，显示更多可用页面

### 模拟开柜流程

1. 认证成功后，在认证页点击 **"申请开柜"**
2. 自动跳转到工具页（柜门打开状态）
3. 观察工具盘点表格的实时更新

### 页面切换方式

- **方式 1**：点击底部导航栏按钮
- **方式 2**：左右滑动屏幕（鼠标拖动）

### 进入维护模式

1. 必须先以管理员身份认证
2. 进入**设置页**
3. 点击 **"进入维护模式"**
4. 系统切换到调试页，可测试各硬件模块

## 🎨 界面预览

### 顶部状态栏
显示：系统名称 | 当前状态 | 登录用户 | 实时时间

### 页面一览

1. **总览页** - 五区工具状态、环境、充电、最近事件
2. **认证页** - NFC/人脸识别、申请开柜、退出登录
3. **工具页** - 五区表格、盘点结果、缺失错放提示
4. **充电页** - 电池盒状态、4个电池位、手动控制
5. **环境页** - 温湿度、风扇、阈值设置、趋势
6. **记录页** - 操作记录、筛选、导出
7. **设置页** - 阈值设置、自动控制、危险操作
8. **调试页** - 硬件状态、单模块测试、日志

### 底部导航栏
根据当前系统状态动态生成，只显示当前允许访问的页面

## 🔧 系统状态说明

| 状态 | 说明 | 可访问页面 |
|------|------|------------|
| STANDBY | 未认证待机 | 总览、认证 |
| USER_AUTHED | 普通用户已认证 | 总览、工具、充电、环境、记录、认证 |
| ADMIN_AUTHED | 管理员已认证 | 总览、认证、工具、充电、环境、记录、设置 |
| CABINET_OPEN | 柜门打开 | 总览、工具、充电、环境、记录 |
| MAINTENANCE | 维护模式 | 调试、设置、总览 |

## 🐛 故障排除

### 问题：无法启动，提示找不到 PyQt5

**解决**：
```bash
# Ubuntu
sudo apt install python3-pyqt5

# pip
pip3 install PyQt5
```

### 问题：界面显示不完整

**解决**：检查分辨率设置，在 `config.py` 中调整或启用全屏模式

### 问题：数据不更新

**解决**：检查终端输出是否有错误信息，确认 MockBackend 已启动
